## Special Notes
### What is needed to Play `Spider-Man`
- A Laptop, Computer or a Desktop with a keyboard
- Specially a Keyboard
- A web browser
- File extractor (normally you can extract with the system software or you can use WinRar and etc.)

- Javascript
